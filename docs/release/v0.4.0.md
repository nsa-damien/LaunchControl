# Release v0.4.0

**Release Date:** 2026-02-17

## Highlights

Adds on-demand execution of launch items directly from the context menu. Right-click any launch agent or daemon and select "Run Now" to trigger it immediately via `launchctl kickstart`.

## What's New

- **Run Now context menu option**: Trigger any launch item on demand using `launchctl kickstart`
- Works with user agents (direct execution) and system agents/daemons (with Touch ID authentication)

No bug fixes in this release.

## Breaking Changes

No breaking changes.

Update to v0.4.0 by downloading the latest release. No configuration changes required.
