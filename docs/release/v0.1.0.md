# Release v0.1.0

**Release Date:** 2026-02-14

## Highlights

First release of LaunchControl — a macOS GUI for managing LaunchAgents and LaunchDaemons. Browse, load/unload, enable/disable launch items with Touch ID authentication for system-level operations. Includes drag-and-drop installation of new user agents.

## What's New

- **Three-category browsing** — User Agents, System Agents, System Daemons with sidebar navigation
- **Full lifecycle management** — Load/unload, enable/disable any launch item
- **Touch ID authentication** — Biometric or password auth for system-level operations
- **Real-time status** — See running, stopped, and loaded state at a glance
- **Search and filter** — Quickly find items across all categories
- **Drag-and-drop install** — Drop .plist files onto the window to install user agents
- **Post-install prompt** — Choose to enable and start agents immediately after install
- **Overwrite protection** — Confirmation dialog when replacing existing agents
- **Delete support** — Remove launch items via context menu
- **Debug diagnostics** — Built-in sheet showing directory access and file counts

## Upgrade Instructions

This is the initial release. Clone the repo, open `LaunchControl.xcodeproj` in Xcode 15+, and build.
